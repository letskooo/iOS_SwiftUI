# iOS_SwiftUI
나모 서비스의 iOS 리포지토리입니다.

## 프로젝트 소개
개인 일정 기록은 물론 친구와 모임을 이뤄 공유할 수 있는 서비스

## 📆 개발 기간
- 참여 개발 기간: 2024.01 ~ 2024.07

## ⚡ 담당 개발 기능
❗commit에서 'letskooo'와 'KoSungmin'이 본인입니다.
- 프로젝트 아키텍처 설계 참여
- 사용자 인증 기능 구현
- 소설 로그인(카카오, 네이버, 애플) 기능 구현
- 카테고리 편집(생성, 수정, 삭제) 화면 및 기능 구현
